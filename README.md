# Profit/Loss Tracker — GitHub Pages

This repository contains a single-page app: `ProfitLoss_Tracker.html`.

How to publish on GitHub Pages

1. Commit and push this repository to GitHub (default branch `main` recommended).
2. Either enable GitHub Pages in repository Settings (serve from the `root` of `main`) or let the included Actions workflow deploy automatically on push to `main`.

Quick push commands:

```bash
git add .
git commit -m "Add site and Pages workflow"
git push origin main
```

The site will be available at `https://<your-username>.github.io/<repo-name>/` once deployed.
